# BAP578-X

## BNB-Native Ops Kit for Non-Fungible Agents

BAP578-X is an operational toolkit for BAP-578 agents on BNB Chain.

### Features
- BSC / opBNB presets
- Metadata compiler
- Gas snapshot
- Multicall batch reader
- Local event indexer

### Usage

npx bap578-x ping --chain bsc
npx bap578-x gas --chain bsc
